    Ficha Técnica: Navegación, Persistencia y Validaciones Automáticas en React#

    Descripción#

    En esta ficha, exploraremos cómo manejar la navegación entre páginas utilizando useState, cómo persistir datos en localStorage para evitar su pérdida al recargar la aplicación y cómo emplear useEffect para validar datos automáticamente antes de avanzar en el flujo de la aplicación.

    Archivos Necesarios y Estructura#

    • App.tsx: Componente principal que gestiona la navegación y la persistencia de datos.
    • localStorage: API del navegador para almacenamiento persistente.
    • useEffect: Hook para validaciones automáticas.

    Código del Proyecto#

    Componente con useState, localStorage y useEffect#

    import React, { useState, useEffect } from "react";

type FormularioDatos = {
    nombre: string;
    edad: string;
};

const App: React.FC = () => {
    const [pagina, setPagina] = useState<string>("bienvenida");
    const [datos, setDatos] = useState<FormularioDatos>(() => {
        const datosGuardados = localStorage.getItem("formularioDatos");
        return datosGuardados ? JSON.parse(datosGuardados) : { nombre: "", edad: "" };
    });

    useEffect(() => {
        localStorage.setItem("formularioDatos", JSON.stringify(datos));
    }, [datos]);

    useEffect(() => {
        if (datos.nombre && datos.edad && pagina === "formulario") {
            setPagina("resumen");
        }
    }, [datos, pagina]);

    return (
        <div>
            {pagina === "bienvenida" && (
                <>
                    <h1>Bienvenido</h1>
                    <button onClick={() => setPagina("formulario")}>Ir al Formulario</button>
                </>
            )}

            {pagina === "formulario" && (
                <>
                    <h1>Formulario</h1>
                    <input
                        type="text"
                        placeholder="Nombre"
                        value={datos.nombre}
                        onChange={(e) => setDatos({ ...datos, nombre: e.target.value })}
                    />
                    <input
                        type="number"
                        placeholder="Edad"
                        value={datos.edad}
                        onChange={(e) => setDatos({ ...datos, edad: e.target.value })}
                    />
                </>
            )}

            {pagina === "resumen" && (
                <>
                    <h1>Resumen</h1>
                    <p>Nombre: {datos.nombre}</p>
                    <p>Edad: {datos.edad}</p>
                </>
            )}
        </div>
    );
};

export default App;

    Conceptos Clave#

    Navegación con useState#

    Podemos cambiar entre diferentes "páginas" controlando un estado.

    const [pagina, setPagina] = useState("inicio");

return (
    <div>
        {pagina === "inicio" && <h1>Inicio</h1>}
        {pagina === "formulario" && <h1>Formulario</h1>}
        <button onClick={() => setPagina("formulario")}>Ir al Formulario</button>
    </div>
);

    Persistencia de Datos con localStorage#

    Guardamos datos en localStorage para que no se pierdan al recargar la página.

    useEffect(() => {
    localStorage.setItem("nombreUsuario", nombre);
}, [nombre]);

    Recuperar datos al cargar:

    const [nombre, setNombre] = useState(() => {
    return localStorage.getItem("nombreUsuario") || "";
});

    Validaciones Automáticas con useEffect#

    Podemos verificar datos y tomar decisiones sin intervención del usuario.

    useEffect(() => {
    if (nombre.length > 3) {
        console.log("Nombre válido");
    }
}, [nombre]);

    Conclusión#

    • Usamos useState para manejar la navegación entre páginas.
    • Persistimos datos con localStorage, evitando pérdidas al recargar.
    • Validamos automáticamente con useEffect, mejorando la experiencia de usuario.
    • Creamos una navegación fluida sin necesidad de un router.
