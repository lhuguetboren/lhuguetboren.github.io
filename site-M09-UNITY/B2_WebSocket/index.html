    Apuntes – Bloque 2: Introducción a WebSocket y chat en Unity#

    Objetivos del bloque#

    • Entender el funcionamiento del protocolo WebSocket.
    • Comparar WebSocket con UDP.
    • Conectar Unity a un servidor WebSocket.
    • Enviar y recibir mensajes en tiempo real.
    • Implementar reconexión y control de errores básicos.

    1. ¿Qué es WebSocket?#

    • Protocolo de comunicación en tiempo real sobre TCP.
    • Permite una conexión persistente entre cliente y servidor.
    • Ideal para chats, juegos, transmisiones, etc.

    🆚 Comparativa: WebSocket vs UDP#

    Característica UDP WebSocket
    Tipo de conexión Sin conexión (no confiable) Persistente (basado en TCP)
    Fiabilidad No garantiza entrega Garantiza entrega
    Velocidad Muy alta Alta
    Casos de uso Juegos, audio en tiempo real Chats, apps web, juegos online

    💻 3. Código base del servidor WebSocket (Node.js)#

    const WebSocket = require('ws');
const wss = new WebSocket.Server({ port: 8080 });

wss.on('connection', (ws) => {
  console.log("Nuevo cliente conectado");

  ws.on('message', (data) => {
    // Enviar a todos los clientes
    wss.clients.forEach(client => {
      if (client.readyState === WebSocket.OPEN) {
        client.send(data);
      }
    });
  });

  ws.on('close', () => {
    console.log("Cliente desconectado");
  });
});

console.log("Servidor WebSocket en ws://localhost:8080");

    🎮 4. Código base del cliente Unity (NativeWebSocket)#

    using NativeWebSocket;

WebSocket websocket;

async void Start() {
    websocket = new WebSocket("ws://localhost:8080");

    websocket.OnOpen += () => Debug.Log("Conectado!");
    websocket.OnMessage += (bytes) => {
        string message = System.Text.Encoding.UTF8.GetString(bytes);
        Debug.Log("Mensaje recibido: " + message);
    };
    websocket.OnClose += () => Debug.Log("Desconectado");
    websocket.OnError += (e) => Debug.Log("Error: " + e);

    await websocket.Connect();
}

public async void EnviarMensaje(string texto) {
    if (websocket.State == WebSocketState.Open) {
        await websocket.SendText(texto);
    }
}
