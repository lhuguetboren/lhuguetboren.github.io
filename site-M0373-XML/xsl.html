Introducción a XSLT

Objetivo

Comprender los fundamentos de XSLT, el lenguaje que permite transformar documentos XML en otros formatos como HTML, XML o texto plano.

1. ¿Qué es XSLT?

XSLT (Extensible Stylesheet Language Transformations) es un lenguaje diseñado para transformar documentos XML. Con XSLT puedes:

  • Convertir XML en HTML para mostrar en un navegador.
  • Filtrar o reordenar datos XML.
  • Generar informes o archivos de texto desde XML.

2. Estructura básica de un archivo XSLT 

<xsl:stylesheet version="1.0"
  xmlns:xsl="http://www.w3.org/1999/XSL/Transform">

  <xsl:output method="html" indent="yes"/>

  <xsl:template match="/">
    <!-- Aquí va el HTML generado -->
  </xsl:template>

</xsl:stylesheet>

🔹 xsl:template match="/" define qué hacer con el nodo raíz del XML.
🔹 xsl:output especifica el tipo de salida: HTML, XML, texto…

3. Ejemplo simple: XML a HTML

XML (libros.xml) 

<catalogo>
  <libro>
    <titulo>1984</titulo>
    <autor>George Orwell</autor>
  </libro>
</catalogo>

XSLT (libros.xsl) 

<xsl:stylesheet version="1.0"
  xmlns:xsl="http://www.w3.org/1999/XSL/Transform">

  <xsl:output method="html" indent="yes"/>

  <xsl:template match="/catalogo">
    <html>
      <body>
        <h1>Libros</h1>
        <xsl:for-each select="libro">
          <p>
            <xsl:value-of select="titulo"/> - <xsl:value-of select="autor"/>
          </p>
        </xsl:for-each>
      </body>
    </html>
  </xsl:template>

</xsl:stylesheet>

Principales instrucciones XSLT

1. <xsl:stylesheet> / <xsl:transform>

Define el documento como hoja de estilo XSLT.
📚 [XSLT xsl:stylesheet](https://www.w3schools.com/xml/el_stylesheet.asp)

<xsl:stylesheet version="1.0"
  xmlns:xsl="http://www.w3.org/1999/XSL/Transform">
  <!-- Plantillas aquí -->
</xsl:stylesheet>

2. <xsl:template match="...">

Define una plantilla que se aplica a nodos que coinciden con el patrón.
📚 [XSLT xsl:template](https://www.w3schools.com/xml/el_template.asp)

<xsl:template match="/">
  <html><body>Contenido</body></html>
</xsl:template>

3. <xsl:value-of select="..."/>

Muestra el contenido de un nodo.
📚 [XSLT xsl:value-of](https://www.w3schools.com/xml/el_value-of.asp)

<xsl:value-of select="catalog/cd/title"/>

4. <xsl:for-each select="...">

Itera sobre un conjunto de nodos.
📚 [XSLT xsl:for-each](https://www.w3schools.com/xml/el_for-each.asp)

<xsl:for-each select="catalog/cd">
  <xsl:value-of select="title"/>
</xsl:for-each>

5. <xsl:if test="...">

Evalúa una condición y aplica el contenido si es verdadera.
📚 [XSLT xsl:if](https://www.w3schools.com/xml/el_if.asp)

<xsl:if test="price > 10">
  <p>Precio alto</p>
</xsl:if>

6. <xsl:choose>, <xsl:when>, <xsl:otherwise>

Estructura condicional múltiple (como if-else).
📚 [XSLT xsl:choose](https://www.w3schools.com/xml/el_choose.asp)

<xsl:choose>
  <xsl:when test="price > 10">Caro</xsl:when>
  <xsl:otherwise>Barato</xsl:otherwise>
</xsl:choose>

Ejercicios de completar:

Ejercicio 1: Mostrar títulos de libros

XML: libros.xml 

<biblioteca>
  <libro>
    <titulo>1984</titulo>
    <autor>George Orwell</autor>
  </libro>
  <libro>
    <titulo>Fahrenheit 451</titulo>
    <autor>Ray Bradbury</autor>
  </libro>
</biblioteca>

XSLT incompleto: 

<xsl:for-each select="<!-- COMPLETAR -->">
  <li><xsl:value-of select="<!-- COMPLETAR -->"/></li>
</xsl:for-each>

Solución: 

<xsl:for-each select="biblioteca/libro">
  <li><xsl:value-of select="titulo"/></li>
</xsl:for-each>

🧪 Ejercicio 2: Tabla de productos con precios

XML: productos.xml 

<tienda>
  <producto>
    <nombre>Manzana</nombre>
    <precio>1.00</precio>
  </producto>
  <producto>
    <nombre>Plátano</nombre>
    <precio>0.80</precio>
  </producto>
</tienda>

XSLT incompleto: 

<xsl:for-each select="<!-- COMPLETAR -->">
  <tr>
    <td><xsl:value-of select="<!-- COMPLETAR -->"/></td>
    <td><xsl:value-of select="<!-- COMPLETAR -->"/></td>
  </tr>
</xsl:for-each>

Solución: 

<xsl:for-each select="tienda/producto">
  <tr>
    <td><xsl:value-of select="nombre"/></td>
    <td><xsl:value-of select="precio"/></td>
  </tr>
</xsl:for-each>

🧪 Ejercicio 3: Mostrar solo usuarios mayores de edad

XML: usuarios.xml 

<usuarios>
  <usuario>
    <nombre>Ana</nombre>
    <edad>22</edad>
  </usuario>
  <usuario>
    <nombre>Juan</nombre>
    <edad>16</edad>
  </usuario>
</usuarios>

XSLT incompleto: 

<xsl:for-each select="usuarios/usuario">
  <xsl:if test="<!-- COMPLETAR -->">
    <li><xsl:value-of select="<!-- COMPLETAR -->"/></li>
  </xsl:if>
</xsl:for-each>

Solución: 

<xsl:for-each select="usuarios/usuario">
  <xsl:if test="edad &gt;= 18">
    <li><xsl:value-of select="nombre"/></li>
  </xsl:if>
</xsl:for-each>
