    Exercici 2: Formulari amb petici贸 POST#

    Objectiu#

    Enviar dades dun formulari amb POST i mostrar la resposta del servidor.

    HTML + JS#

    <h2>Formulari</h2>
<form id="formulari">
  <input type="text" name="nom" placeholder="Nom" required>
  <input type="number" name="edat" placeholder="Edat" required>
  <button type="submit">Enviar</button>
</form>

<div id="resposta-post"></div>

<script>
  const form = document.getElementById('formulari');

  form.addEventListener('submit', function(e) {
    e.preventDefault(); // Evitem recarregar la pgina

    const dades = {
      nom: form.nom.value,
      edat: form.edat.value
    };

    fetch('servidor.php', {
      method: 'POST',
      headers: {
        'Content-Type': 'application/json'
      },
      body: JSON.stringify(dades)
    })
    .then(res => res.json())
    .then(resposta => {
      document.getElementById('resposta-post').innerText =
        `Nom: ${resposta.nom}, Edat: ${resposta.edat}`;
    })
    .catch(error => {
      console.error('Error:', error);
    });
  });
</script>

    servidor.php (exemple bsic)#

    <?php
header('Content-Type: application/json');
$entrada = json_decode(file_get_contents('php://input'), true);
echo json_encode($entrada);

    Explicaci贸#

    • Senvia un JSON amb fetch().
    • El servidor llegeix i retorna el mateix JSON.
    • Mostrem la resposta al div.

    Exercici 3: Enviament de fitxer amb FormData#

    Objectiu#

    Enviar un fitxer al servidor i mostrar la seva informaci贸.

    HTML + JS#

    <h2>Upload Fitxer</h2>
<input type="file" id="fitxer">
<button id="envia">Enviar</button>
<div id="resposta-fitxer"></div>

<script>
  document.getElementById('envia').addEventListener('click', function() {
    const input = document.getElementById('fitxer');
    const arxiu = input.files[0];

    if (!arxiu) {
      alert('Selecciona un fitxer abans denviar.');
      return;
    }

    const formData = new FormData();
    formData.append('fitxer', arxiu);

    fetch('upload.php', {
      method: 'POST',
      body: formData
    })
    .then(res => {
      if (!res.ok) throw new Error('Error del servidor');
      return res.json();
    })
    .then(info => {
      document.getElementById('resposta-fitxer').innerHTML = `
        Nom: ${info.nom} <br>
        Mida: ${info.mida} bytes <br>
        Tipus: ${info.tipus}
      `;
    })
    .catch(error => {
      document.getElementById('resposta-fitxer').innerText = 'Error: ' + error.message;
    });
  });
</script>

    upload.php (exemple funcional)#

    <?php
header('Content-Type: application/json');

if (isset($_FILES['fitxer'])) {
    $fitxer = $_FILES['fitxer'];
    echo json_encode([
        'nom' => $fitxer['name'],
        'mida' => $fitxer['size'],
        'tipus' => $fitxer['type']
    ]);
} else {
    http_response_code(400);
    echo json_encode(['error' => 'No sha rebut cap fitxer']);
}

    Explicaci贸#

    • FormData() empaqueta el fitxer.
    • El servidor rep $_FILES['fitxer'] i retorna dades.
    • Gestionem errors si no hi ha fitxer o el servidor falla.
