    Introducció a WebSocket amb Node.js i JavaScript#

    WebSocket és un protocol que permet comunicació bidireccional en temps real entre el client (navegador) i el servidor. A diferència de HTTP, WebSocket manté una connexió oberta, ideal per aplicacions com xats, jocs o notificacions.

    sequenceDiagram
    participant Navegador1
    participant Navegador2
    participant ServidorWS as Servidor WebSocket

    Note over Navegador1,ServidorWS: Connexió inicial
    Navegador1->>ServidorWS: WebSocket.connect("ws://localhost:8089")
    ServidorWS-->>Navegador1: Connexió acceptada

    Note over Navegador2,ServidorWS: Segona connexió
    Navegador2->>ServidorWS: WebSocket.connect("ws://localhost:8089")
    ServidorWS-->>Navegador2: Connexió acceptada

    Note over Navegador1: Envia un missatge
    Navegador1->>ServidorWS: socket.send("Hola servidor!")
    ServidorWS-->>Navegador2: Missatge d'un altre client: Hola servidor!

    Note over Navegador1,ServidorWS: Tancament de connexió
    Navegador1-->>ServidorWS: Connexió tancada
    ServidorWS-->>ServidorWS: Elimina la connexió del llistat

    Instal·lació completa de Node.js i WebSocket#

    🔗 Pàgina oficial:#

    ➡️ https://nodejs.org

    Descarrega i instal·la la versió recomanada (LTS) segons el teu sistema operatiu.

    2️⃣ Verifica la instal·lació#

    Obre una terminal o consola i escriu:

    node -v      # Mostra la versió de Node.js
npm -v       # Mostra la versió de npm (Node Package Manager)

    3️⃣ Crea un projecte nou#

    mkdir servidor-websocket
cd servidor-websocket
npm init -y

    4️⃣ Instal·la el paquet websocket#

    npm install websocket

    📦 Això afegirà el mòdul a node_modules/ i al fitxer package.json.

    5️⃣ Crea el servidor WebSocket#

    Crea un fitxer server.js amb aquest contingut:

    const http = require('http');
const WebSocketServer = require('websocket').server;

const server = http.createServer((req, res) => {
  res.end("Servidor WebSocket actiu");
});

const wsServer = new WebSocketServer({
  httpServer: server,
  autoAcceptConnections: false
});

const connexions = [];

wsServer.on('request', (request) => {
  const connection = request.accept(null, request.origin);
  connexions.push(connection);
  console.log("Client connectat");

  connection.on('message', (message) => {
    console.log("Missatge rebut:", message.utf8Data);
    connexions.forEach((connexioClient) => {
      if (connexioClient !== connection) {
        connexioClient.sendUTF("Missatge d'un altre client: " + message.utf8Data);
      }
    });
  });

  connection.on('close', () => {
    console.log("Connexió tancada");
    const index = connexions.indexOf(connection);
    if (index !== -1) connexions.splice(index, 1);
  });
});

server.listen(8089, () => {
  console.log("Servidor WebSocket escoltant al port 8089");
});

    6️⃣ Executa el servidor#

    node server.js

    ✅ Característiques:#

    • Accepta múltiples connexions.
    • Reenvia missatges entre clients.
    • Gestiona desconnexions.

    💻 Client WebSocket en JavaScript (navegador)#

    <script>
  const socket = new WebSocket("ws://localhost:8089");

  socket.onopen = function () {
    console.log("Connectat al servidor WebSocket");
    socket.send("Hola servidor!");
  };

  socket.onmessage = function (event) {
    console.log("Missatge rebut del servidor:", event.data);
  };

  socket.onclose = function () {
    console.log("Connexió tancada");
  };

  socket.onerror = function (error) {
    console.error("Error WebSocket:", error);
  };
</script>

    ✅ Comportament:#

    • Es connecta al servidor WebSocket.
    • Envia i rep missatges en temps real.
    • Mostra la comunicació per consola.

    Resum#

    • WebSocket permet comunicació en temps real sense tornar a fer peticions contínues.
    • Ideal per xats, jocs multiplayer o aplicacions en viu.
    • Funciona amb un servidor WebSocket (Node.js) i un client (JavaScript).
