    Uso de await en JavaScript y React#

    🔹 ¿Cuándo usar await?#

    Usa await cuando llamas a una función asíncrona que devuelve una promesa, como: - fetch() - axios.get() - funciones marcadas como async - operaciones con bases de datos, APIs, timeouts, etc.

    🔹 ¿Por qué usar await?#

    • Para esperar a que se resuelva una promesa sin callbacks ni .then().
    • Hace que el código sea más limpio, legible y secuencial.
    • Facilita el manejo de errores con try/catch.

    🔹 ¿Cómo usar await?#

    🔸 En JavaScript clásico:#

    async function cargarDatos() {
  try {
    const res = await fetch('https://api.example.com/info');
    const datos = await res.json();
    console.log(datos);
  } catch (err) {
    console.error('Error al cargar:', err);
  }
}

    🔸 En React (por ejemplo, en useEffect):#

    useEffect(() => {
  const cargar = async () => {
    const res = await fetch('/api/usuarios');
    const json = await res.json();
    setUsuarios(json);
  };
  cargar();
}, []);

    🚨 Reglas clave:#

    • Solo puedes usar await dentro de una función async.
    • await pausa la ejecución de esa función hasta que se resuelva la promesa.
    • No bloquea el resto del código (¡no congela la app!).

    🟢 Ejemplos de await con POST#

    En JavaScript clásico#

    async function enviarDatos() {
  const datos = { nombre: 'Juan', edad: 30 };

  try {
    const res = await fetch('https://api.example.com/usuarios', {
      method: 'POST',
      headers: {
        'Content-Type': 'application/json'
      },
      body: JSON.stringify(datos)
    });

    const resultado = await res.json();
    console.log('Respuesta:', resultado);
  } catch (error) {
    console.error('Error al enviar:', error);
  }
}

    En React (evento de botón)#

    import { useState } from 'react';

function Formulario() {
  const [mensaje, setMensaje] = useState('');

  const enviar = async () => {
    const datos = { nombre: 'Ana', email: 'ana@example.com' };

    try {
      const res = await fetch('/api/contacto', {
        method: 'POST',
        headers: {
          'Content-Type': 'application/json'
        },
        body: JSON.stringify(datos)
      });

      const resultado = await res.json();
      setMensaje('Enviado correctamente: ' + resultado.status);
    } catch (err) {
      setMensaje('Error al enviar: ' + err.message);
    }
  };

  return (
    <div>
      <button onClick={enviar}>Enviar datos</button>
      <p>{mensaje}</p>
    </div>
  );
}

    📌 Resumen rápido:#

    • method: 'POST' define que es un envío.
    • headers indica el tipo de contenido (JSON, por ejemplo).
    • body lleva los datos (usualmente con JSON.stringify()).
    • Usa await para esperar la respuesta y luego await res.json() para extraerla.
